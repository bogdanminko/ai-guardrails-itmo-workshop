# AI Guardrails для Мегашколы ИТМО

Всем привет, здесь вы сможете найти [ноутбук](guardrails.ipynb) и [презентацию](guardrails-concept.pdf) с моего выступления про Guardrails

Здесь представлены практические примеры построения системы защиты LLM-агента от утечки персональных данных, промпт-инъекций и вредоносного контента. Вы научитесь:
- Детектировать и редактировать PII (имена, телефоны, адреса) с помощью GLiNER
- Блокировать токсичный контент и промпт-атаки с помощью RuBERT-toxicity и GliClass
- Строить многослойную guardrail-систему для production
- Мониторить работу агента через Langfuse

Кейс: AI-ассистент службы доставки еды с инструментами для управления заказами, получения информации о курьерах и выдачи промокодов.


## Старт
### Установка зависимостей
Склонируйте репозиторий, или сохраните себе ноутбук и `pyproject.toml` 

Если вы используете `uv` вы можете синхронизировать зависимости:
```sh
uv sync
```
### Запуск ноутбука
Затем вы можете уверенно открывать и запускать ноутбук: [guardrails.ipynb](guardrails.ipynb)



## Установка Langfuse 
В рамках практики мы обсуждали Langfuse, вы можете поставить его следующим образом и запустить полностью локально:

```
# Get a copy of the latest Langfuse repository
git clone https://github.com/langfuse/langfuse.git
cd langfuse

# Run the langfuse docker compose
docker compose up
```
